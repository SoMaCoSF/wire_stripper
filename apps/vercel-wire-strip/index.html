wire_stripper
Public gist

Control your firehose

wire_stripper is a local perimeter compiler. It fuses the MCP-in-browser insight (tool router as choke point) with DMBT attribution (IP→ASN→prefix) and Privacy Proxy semantics (cookies/fingerprint) into one SQLite truth store. 


      

      

        
Runbook (local)

        
python -m wire_stripper --root .\\data db init
python -m wire_stripper --root .\\data etl --profile default import-all

        

          This Vercel build is static and shows the spec/diagrams. The live capture pipeline (CDP network + policy preflight)
          is implemented locally and will be surfaced via OTel/OTLP and/or an API in a later deploy.
